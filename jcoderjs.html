jCoder.js

A Javascript plugin for hiding script tags in DOM


jCoder.js is a light weight plugin for protecting and hiding your script tags in the DOM from other people using your web app. Sometimes when people hit the "view page source" in the browser, most javascript codes are present. You will notice that sometimes not all javascript codes are there but can be located when you hit inspect element browser functionality. The use of this plugin is simple, when people need to view "inspect element" to see your script tags, they will not find most of your Javascript codes.This plugin automatically hides tags including it's content when you are viewing inspect element in the browser.Do not worry cause your Javascript tags will still be working. This is already tested in chrome and firefox inspect element including fire bug

Reasons for using this pugin:

1. There are cases where the developer will need to hide script tags in DOM when a browser inspect element is used

2. Deep Security

3. Data Encapsulation is never enough to secure your web app

Just include this to your < script src='jcoderjs.js' > < /script > and your good to go
